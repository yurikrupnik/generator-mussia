# React-boilerplate  

## Get started

```
npm run start:mongo
```

Start MongoDB instance on local machine

For more details run `$ docker --help`

Run App:
```
npm start
```

## Features
- Webpack 4
- Babel
- React - context api as state manager
- React-router
- Server side rendering
- Express
- Dynamic imports
- MongoDB - Api
- Auth
- Jest testing
- Eslint checking
- MongoDB docker 
- Sass
